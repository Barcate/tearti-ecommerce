# Tearti Back-end

Criar banco de dados:

```sh
npm run create_db
```

Popular banco de dados com informações placeholder:

```sh
npm run populate_db
```

Remover banco de dados:

```sh
npm run delete_db
```

Remover banco de dados e criar novamente:

```sh
npm run reset_db
```

Para testar:

```sh
npm run test
```
